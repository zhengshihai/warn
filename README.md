# ========学生晚归预警系统========
## 项目背景
系统采用B/S架构，基于Spring MVC框架构建三层架构（表现层、业务逻辑层、数据访问层）。后端采用Spring Framework 6.0 + MyBatis 3.5实现MVC分层与ORM映射，MySQL 8.0作为主数据库，Redis用于缓存与分布式锁。实时通信采用WebSocket实现一键报警的实时定位与视频流传输，集成腾讯地图API实现IP定位与位置展示。视频处理采用JavaCV/FFmpeg实现视频格式转换与HLS切片以支持流媒体播放，前端采用JSP + AJAX实现动态交互，ECharts实现数据可视化，Apache POI实现Excel报表导出。安全机制基于Session + AOP的权限控制。通过以上技术路线，系统能够实现高效、稳定、可扩展的晚归管理功能，为高校智慧校园建设提供有效支撑。
